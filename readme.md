# mapcode

> Find [mapcodes](http://www.mapcode.com/aboutmc.html) in a string

## Install

```sh
npm install --save mapcode
```

## Usage

```js
import mapcode from 'mapcode';

mapcode('Find these FRA 4J.Q2 Hawaii ZSR.3J mapcodes');
//=> ['FRA 4J.Q2', 'Hawaii ZSR.3J']
```
